# CHATTER

Real-time Chat Application built using MERN stack and Socket.IO. This application enables instant messaging between users with secure authentication and efficient data handling.

# Technology Used:

React for frontend, Node.js and Express.js for backend APIs, MongoDB for database, Socket.IO for real-time communication, JWT for authentication.

# Additional Features:

Includes real-time messaging, secure user authentication, responsive UI, scalable backend architecture, and efficient message handling.
